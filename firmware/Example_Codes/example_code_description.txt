This folder contains the different example codes for getting started with STM32. (Will add the pins and phcircuit photo for setting up later on)


Blinky: Contains code for blinking the green LED already present on the STM32 board. 

LCD: Contains code for displaying on LCD.

ADC: Contains code for Analog to digital converter.

WS2812_LED: Code to control RGB lights using PWM.

WIFI LED Blink: Code to blink on LED using esp8266 wifi to connect to a access point and sending an html page. Can also be used to make ESP8266 act as an access point
